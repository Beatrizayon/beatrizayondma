About Me

Artist Statement:

I am Beatriz Ayon Tapia, a first-generation Mexican American college student at San Jose State. I have always used art as a means of self expression -- a way to process, communicate, and connect to the world around me. Part of my practice is exploring different creative mediums and programs. Here at San Jose State University, the Digital Media Arts (DMA) program has enabled me to collaborate with various media and computer programs to create visual works without limits.

Art 74

  • Minecraft 3D Print I first built my model within a Minecraft world using blocks to construct an Indian Ring Neck Parrot. As I have a parrot at home, I thought it'd be a cute souvenir to have with me. I then extracted my build as an STL file using Mineways to then 3D print my final model.
  • Mini-game: Catch the Fruits! Catch the fruits in a fun mini-game made through p5.js.
  • Blue Burrow Cafe
    Work a shift at the Blue Burrow Cafe! Serve the customers the correct order to prevent them from glitching out. This game contains backgrounds, characters, and props drawn by me. I hand coded the project using html and image mapping for the user interactivity.

Illustrator

  • Artist Tour Advertisement For this assignment we were to design a billboard, merchandise, and a poster for an Artist Tour. My music artist of choice was DPR IAN. Through this assignment I was able to play with cohesive yet different compositions to suit the type of product.
  • Self Portrait Grayscale vector portrait in Adobe Illustrator, built with flat tonal shapes and directional light.
  • Dreamscape For this assignment we were to make a dreamlike illustration. When sketching out ideas I wanted to create a colorful galaxy like atmosphere. I was able to incorporate my personal style using Illustrator's tools.
